First of All:
1. Download npm, Git, hexo-cli
2. Clone the blog repo to the local evn
3. ```javascrpt git branch -a ``` to see all branches
4. git checkout [myblog-branch]
5. hexo new post 'Post Name' to create a new post
6. Down!
